# 19-100-0105: INTRODUCTION TO COMPUTING

## Course Objectives
1. To introduce the fundamentals of computers and programming.
2. To develop problem-solving and algorithmic thinking.
3. To teach structured programming using C.

## Course Outcomes
* **CO 1:** Understand computer organization and operation.
* **CO 2:** Write simple C programs using loops and functions.
* **CO 3:** Apply programming logic to solve basic engineering problems.

## Course Content

### Module I – Basics of Computers
* Components of a computer, operating systems, software types.

### Module II – Algorithms and Flowcharts
* Problem-solving techniques, structured approach to programming.

### Module III – Introduction to C Programming
* Data types, operators, control structures, arrays.

### Module IV – Functions and Pointers
* Functions, recursion, pointers, and dynamic memory.

### Module V – Structures and File Handling
* Structures, unions, file I/O, and real-world examples.

## References
1. Kernighan & Ritchie, *The C Programming Language*, Prentice Hall.
2. E. Balagurusamy, *Programming in ANSI C*, McGraw Hill.
