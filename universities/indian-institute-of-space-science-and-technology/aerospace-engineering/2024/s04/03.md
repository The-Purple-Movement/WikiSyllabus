# ae222c: aerospace-propulsion-1
  - (AE222C: Aerospace Propulsion 1)

## Course Objectives
* To understand air breathing jet engine types and working principles
* To analyze thrust and performance parameters of jet engines
* To study thermodynamic cycle analysis of propulsion systems
* To examine components like inlets, combustors and nozzles
* To learn about turbomachinery concepts and hypersonic propulsion

## Course Content

### Module 1: Introduction and Engine Fundamentals
* Introduction to aerospace propulsion systems
* Air Breathing Jet Engine Types and classification
* Thrust generation principles
* Performance parameters and metrics
* Basic propulsion equations

### Module 2: Thermodynamic Analysis
* Air Breathing Jet Engine Thrust calculations
* Performance parameter analysis
* Thermodynamic Cycle Analysis methods
* Brayton cycle and variations
* Efficiency calculations for propulsion systems

### Module 3: Engine Component Aerothermodynamics
* Aerothermodynamics of Inlets and diffusers
* Combustors and Afterburners design
* Nozzle aerothermodynamics and performance
* Component matching and integration
* Performance optimization techniques

### Module 4: Turbomachinery Fundamentals
* Basic Turbomachinery Concepts
* Axial Compressors principles and design
* Axial Flow Turbines operation
* Stage analysis and performance
* Turbomachinery efficiency considerations

### Module 5: Advanced Propulsion Systems
* Aircraft Gas Turbine Engine Off-Design Analysis
* Introduction to Hypersonic Air Breathing Propulsion
* Overview of UAV Propulsion Systems
* Emerging propulsion technologies
* System integration and control aspects
