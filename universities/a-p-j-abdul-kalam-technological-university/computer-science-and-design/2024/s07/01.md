# CXT401: User Interface Software and Technology

## Course Objectives
* Learn principles and technologies for designing effective user interfaces.
* Develop skills in interaction design, layout, navigation, and prototyping using Bootstrap5.

## Course Outcomes
* **CO1:** Summarize interaction design principles and user behavior patterns.
* **CO2:** Explain navigational models and layout patterns.
* **CO3:** Apply design patterns and Bootstrap5 components.
* **CO4:** Develop practical UI prototypes.
* **CO5:** Analyze and improve user experience.

## Course Content

### Module 1: Foundations of Interaction Design
* User behavior, content organization, information architecture.

### Module 2: Navigation and Layout
* Layout systems, navigation patterns, usability principles.

### Module 3: Bootstrap and UI Frameworks
* Bootstrap5 structures and components.

### Module 4: Visual Design and Prototyping
* Page layouts, wireframing, prototyping tools.

### Module 5: Evaluation and UX Analysis
* Usability testing, A/B testing, iterative improvement.

## References
- Alan Dix et al., Human-Computer Interaction.
- Ben Shneiderman, Designing the User Interface.
- Bootstrap Documentation.
