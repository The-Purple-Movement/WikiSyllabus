# EST102: Programming in C

## Course Objectives

The course aims to equip students with:
* A thorough understanding of the fundamentals of programming.
* Problem-solving skills using algorithms and flowcharts.
* Proficiency in the C programming language for application development.
* The ability to debug, test, and execute programs.

## Course Content

### Module - 1 (Introduction to Programming)
* Introduction to algorithms, flowcharts, and pseudo code.
* Basics of programming languages: machine, assembly, high-level.
* Introduction to C language: structure of a C program, compiling, linking, and executing.

### Module - 2 (Data Types, Operators, and Expressions)
* Variables, constants, and keywords.
* Data types: int, float, char, double, void.
* Operators: arithmetic, relational, logical, assignment, increment/decrement, conditional, bitwise.
* Type conversion and precedence of operators.

### Module - 3 (Control Statements)
* Decision-making: if, if-else, nested if-else, switch-case.
* Looping: while, for, do-while loops.
* Break, continue, goto statements.

### Module - 4 (Functions and Arrays)
* Functions: definition, declaration, arguments, return values, recursion.
* Scope and lifetime of variables.
* Arrays: one-dimensional, two-dimensional, initialization, and operations.
* Strings: declaration, initialization, string handling functions.

### Module - 5 (Pointers and Structures)
* Pointers: declaration, initialization, pointer arithmetic.
* Pointers and arrays, pointers and functions.
* Structures: declaration, definition, initialization, accessing members.
* Structures and pointers, arrays of structures.
